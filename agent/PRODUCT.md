# Product

## Register

product

## Users

SCADA/IoT operators and engineers who need AI-powered vision analysis, system monitoring, and operational support through a chat interface.

## Product Purpose

Vision Assistant is an internal operations tool for analyzing SCADA dashboards, diagnosing errors, and managing reports. The primary interface is a chat-first experience powered by a multi-agent LangGraph system that routes requests to Vision, Chat, or Report agents via SSE streaming.

## Brand Personality

Precise, technical, and trustworthy. Feels like professional operations software, not a consumer AI toy.

## Anti-references

- No gradient text or glassmorphism
- No marketing-style hero sections
- No one-note dark blue interface
- No decorative blobs or meaningless ornaments

## Design Principles

1. **Utility first** — Every element serves an operational purpose
2. **Status visibility** — Connection state, agent routing, and loading are always clear
3. **Fast feedback** — Streaming responses, instant copy, clear error states
4. **Professional restraint** — No decorative noise; clean, scannable layouts

## Accessibility & Inclusion

WCAG AA target. Clear focus states, sufficient contrast ratios, reduced motion support.
